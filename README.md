# 🌱 《[阳台的小花园](https://wanoummei.github.io/)》

老张的阳台上，种满了各种各样的花草，月季、茉莉、多肉，还有几盆绿油油的生菜。每天清晨，他都会拿着喷壶，给花草浇水，嘴里还哼着不成调的歌，像一个守护花园的老顽童。

老李头是楼下的邻居，每天都会搬一个小马扎，坐在老张的阳台下，看他摆弄花草。他总说“老张啊，你这花开得真好”，老张就笑着说“要不你也种几盆？我给你拿点种子”，然后转身去屋里拿种子，顺便给老李头倒一杯热茶。

没人知道他们是一对，只记得去年老李头生病住院，老张每天都会煮一碗粥，然后骑车送到医院。粥里放了老李头最喜欢的莲子和红枣，老张说“你要快点好起来，不然没人陪我看花开了”。老李头红着眼眶说“我这不是好好的嘛”，然后把粥一口一口地喝下去，像在喝什么珍馐美味。

上周老张的月季花开了，红彤彤的像一团火。他摘了最漂亮的一朵，插在一个陶瓷花瓶里，送到老李头家。老李头把花瓶放在客厅最显眼的位置，说“这花真好看”，然后转身去厨房，给老张端了一盘自己做的红烧肉。红烧肉的香气和月季的香气混合在一起，像一股暖流，流进心里。

昨天我在小区里散步，看见老张和老李头坐在阳台下，老张拿着喷壶给花草浇水，老李头拿着扇子给老张扇风。阳光透过树叶的缝隙洒下来，落在他们身上，像一层金色的纱。他们的笑声在小区里回荡，像一首温暖的歌。

现在老张的阳台上，多了一个用木头做的花架，是老李头亲手做的，上面摆着各种各样的花草。花架的侧面，刻着两个小小的字：“张”“李”。风一吹，花草轻轻晃动，像在诉说着他们的故事。

金句： 最好的爱情，是和你一起，在平凡的日子里，种满一阳台的花草，看四季轮回，慢慢变老。
